# Docker Dagger Module

Reusable Dagger-native Docker and OCI image helpers for CI workflows.

## Status

This module is being introduced incrementally. The initial skeleton provides
module metadata, Python package layout, and a neighboring Dagger test module.
Build, smoke check, registry auth, and publish APIs are added by later tasks in
the OpenSpec change.

## Local Paths

- Module source: `src/docker/`
- Dagger test module: `tests/`
- Public facade: `src/docker/main.py`
